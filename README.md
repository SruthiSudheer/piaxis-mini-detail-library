# piaxis-mini-detail-library

## Setup Instructions

1. Create PostgreSQL database:
   
    *Name: Piaxis_local*

3. Create Table and insert seed data by running:
   
   *cd backend*<br>
   *npm i*<br>
   *node test-db.js*

5. Backend setup:   

   *node index.js*

7. Frontend:
   
   Open index.html in browser
